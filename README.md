# yummy
The project is web application - rest-api in spring with Angular client.
Yummy is a site that allows users find perfect recipe, comment and rate other people ideas

## technologies
- Spring Boot
- Angular
- PostgreSQL
- Firebase
- HTML, CSS

## features
- login/register
- searching for recipes
- recipes filtering by dynamically loaded categories, ingredients, cooking time or titles
- pagination
- profile editing
- rating and comments sections for each recipe


## Security
Project provides authentication and authorization with JWT Spring Security.
There are parts of application available for not logged users, and parts that require specific
permissions.
> User types:
- not logged in
- user
- admin
